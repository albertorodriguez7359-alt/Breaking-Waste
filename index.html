Sobre nuestro proyecto

Breaking Waste es un sistema inteligente de monitoreo de residuos que utiliza un sensor infrarrojo FC-51 conectado a un ESP32 para medir el nivel de llenado de un contenedor y mostrar la información en una aplicación web en tiempo real.

Características del sistema

ESP32

Microcontrolador encargado de enviar los datos del sensor.

Sensor FC-51

Permite detectar obstáculos, medir distancias y rastrear líneas.

Aplicación Web

Interfaz donde se observa el estado del contenedor.
