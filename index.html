Sobre nuestro proyecto

Breaking Waste es un sistema inteligente de monitoreo de residuos que utiliza sensores infrarrojos FC-51 conectados a un ESP32 para detectar el nivel de llenado de un contenedor y mostrar la información en una aplicación web en tiempo real.

Características del sistema

ESP32

Microcontrolador encargado de leer los sensores y enviar la información mediante una red WiFi.

Sensor FC-51

Sensor infrarrojo utilizado para detectar cuándo los residuos alcanzan un nivel determinado.

Aplicación web

Interfaz donde el usuario puede consultar el estado y nivel del contenedor.
